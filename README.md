# Bhaswanth Vommena — Professional Portfolio

Welcome to the source repository for my personal portfolio. I am a **Data Professional, Entrepreneur, and AI Product Builder** specializing in building modern web applications, interactive data dashboards, and AI-powered interfaces.

This portfolio serves as a showcase of my technical skills, commercial projects, and hands-on experience in frontend engineering, data analysis, and AI application design.

---

## 🚀 Key Features

- **Premium 3D Visuals & Shaders**: Integrated WebGL interactive canvas backdrops built with React Three Fiber (`@react-three/fiber`) and `@react-three/drei`.
- **Interactive Project Showcase**:
  - 📊 **Global Supply Chain Dashboard**: Interactive data visualization dashboard highlighting late-delivery risk analysis and global shipping bottlenecks, built using custom Recharts visualizations.
  - 🤖 **Enterprise RAG Chatbot**: A simulated agentic search-and-chat interface designed to demonstrate AI UX patterns, pgvector search stages, chunk retrieval mechanisms, and token-streaming rendering.
  - 💼 **Commercial Deployments**: Highlights live production web projects built for client companies, featuring custom savings calculators, responsive menus, and advanced SEO/Analytics.
- **Fluid & Responsive Motion**: Powered by Framer Motion to provide high-performance page entries, card hover micro-animations, and smooth page routing transitions.
- **Modern Tech Stack Display**: Visually maps out my toolbelt spanning Languages, Frameworks, AI IDEs, and specialized developer tools.

## 🛠️ Technology Stack

- **Framework**: [Next.js (App Router)](https://nextjs.org/)
- **Core Library**: [React 19](https://react.dev/)
- **Styling**: Vanilla CSS Modules (modular, scoped CSS layout control)
- **3D Graphics**: [Three.js](https://threejs.org/) / `@react-three/fiber` & `@react-three/drei`
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [Recharts](https://recharts.org/)
- **Icons**: [Lucide React](https://lucide.dev/)

## ⚙️ Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (version 18+ is recommended).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bhaswanthvommena/My-portfolio.git
   cd My-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.

---

## 📦 Building for Production

To create an optimized production build:

```bash
npm run build
```

This compiles the Next.js pages, statically generates HTML files, and bundles your CSS modules and React components. To preview the production build locally:

```bash
npm run start
```
